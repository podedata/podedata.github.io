# Case Coercial [Análise de Pessoas] - Podedata

## Informações Básicas
- Nome do Projeto: Análise dos Recursos Humanos
- Indústria/Segmento: Indústria
- Data/Período: 2026
- Ferramentas Utilizadas: (Power BI, Pentaho PDI)

## Contexto e Desafio
1. Qual era a situação inicial do cliente?
-> Análise e consolidação manual por etapas com grande volume de relatórios individuais para o contexto.
2. Quais eram os principais problemas/dores?
-> Coleta extremamente demorada dos dados para consolidação, já que dependia de outras áreas para envio dos dados.
-> Tempo, complexidade na lógica de consolidação, visual ultrapassado, possibilidade de erros na consolidação das informações.
3. Qual era o impacto no negócio?
-> Médio/Alto impacto

## Solução Implementada
1. O que foi feito?
	- Implantação e configuração de um ambiente próprio para dados, o que gera segurança, performance, integridade e confiabilidade na entrega do resultado.
	Partimos de um ambiente de dados zero para um ambiente controlado, gerenciável, eficiente com diagnóstico em tempo praticamente real, com performance excelente na geração do resultado,
	o que fez o cliente renovar contrato para adiantar outros projetos de dados programados para datas futuras.
2. Principais entregas:
	- Ambiente performático de extração de dados necessários para a composição do fato a ser analisado
	- Redução de tempo para a entrega do objetivo
	- Performance, qualidade e integridade dos dados
	- Visual desenhado conforme o gosto e necessidade do cliente para a análise do contexto
3. Tecnologias/Metodologias:
	- Mapeamento de demanda, planejamento
	- Levantamento e mapeamento dos dados
	- ETL e Engenharia de Dados
	- Modelagem e desenvolvimento de DAX
	- Desenvolvimento da Visualização e Storytelling
	- Entrega ao Cliente e Governaça
4. Timeline do projeto:
	- Sustentação

## Resultados e Impacto
1. Resultados quantitativos:
	- Redução de aproximadamente 75% do tempo manual
2. Resultados qualitativos:
	- Tomada de decisão rápida e confiável
	- Melhor compreensão do contexto
	- Cultura de Dados
	- Alinhamento estratégico
